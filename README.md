## Developer Tools

| Command         | Description                          |
|-----------------|--------------------------------------|
| `make format`   | Format code with Black               |
| `make lint`     | Lint + autofix with Ruff             |
| `make test`     | Run tests with pytest                |
| `make security` | Run Bandit and Safety checks         |
| `make all`      | Run all of the above                 |
